现在 `fetchall()` 拿到的是 **tuple**，要变成 **dict** 有两种最常用做法：

------

## 方式 1（推荐）：用 `DictCursor`，查询出来直接就是字典

```python
import pymysql

conn = pymysql.connect(
    host="127.0.0.1",
    user="root",
    password="xxx",
    database="xxx",
    charset="utf8mb4",
    cursorclass=pymysql.cursors.DictCursor,  # 关键
)

with conn.cursor() as cur:
    cur.execute("SELECT dept_id, parent_id, dept_name, leader FROM sys_dept LIMIT 5;")
    rows = cur.fetchall()
    print(rows[0])  # {'dept_id': 100, 'parent_id': 0, 'dept_name': '若依科技', 'leader': '若依'}
conn.close()
```

------

## 方式 2：保持默认 cursor，用 `cursor.description` 手动把 tuple 转 dict

```python
with conn.cursor() as cur:
    cur.execute("SELECT dept_id, parent_id, dept_name, leader FROM sys_dept LIMIT 5;")
    rows = cur.fetchall()

    cols = [desc[0] for desc in cur.description]  # 列名
    dict_rows = [dict(zip(cols, row)) for row in rows]

    print(dict_rows[0])
```

------

## 只想把“单条 tuple”转 dict（比如你已经有一条 row）

```python
cols = ["dept_id", "parent_id", "dept_name", "leader"]
row = (100, 0, "若依科技", "若依")
d = dict(zip(cols, row))
print(d)
```

------

如果你后面要做组织树（parent_id 关联），我建议你直接用 **方式1 DictCursor**，后面写起来最顺。
