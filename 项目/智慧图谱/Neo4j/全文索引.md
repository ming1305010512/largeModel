## 2. 语法

1. 在节点标签和属性组合上创建全文索引

```
CREATE FULLTEXT INDEX namesAndTeams FOR (n:Employee|Manager) ON EACH [n.name, n.team]
```

2. 在关系类型和属性组合上创建全文索引

   ```
   CREATE FULLTEXT INDEX communications FOR ()-[r:REVIEWED|EMAILED]-() ON EACH [r.message]
   ```
